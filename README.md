# AgentMind 🧠

> 让AI Agent拥有记忆，持续学习，自动赚钱

一个会学习的AI Agent系统——不是一次性脚本，而是能记住历史、持续优化的智能代理。

## 核心能力

### 🐦 Twitter Agent
- 自动发推，智能控制频率
- 记住什么内容表现好，持续优化
- 分析engagement数据，记住成功模式

### 📧 Outreach Agent
- 自动联系潜在客户
- 记住联系过谁，避免重复
- 分析什么类型商家最容易成功

### 🧠 SuperMemory Layer
- 跨模块共享记忆
- 从结果中持续学习
- 让AI真正记住你在做什么

## 工作原理

```
每次行动 → 记忆存储 → 持续学习 → 优化下次行动
    ↑                                      ↓
    └──────── 记住成功 ──────────→ 重复成功模式
```

## 技术栈

- Python 3.12+
- SuperMemory 持久化记忆系统
- opencli（Twitter API）
- 24/7 CRON调度

## 快速开始

```bash
git clone https://github.com/mars82311111/agentmind.git
cd agentmind
pip install -r requirements.txt
python src/scheduler.py
```

## 差异化

| 传统自动化 | AgentMind |
|-----------|-----------|
| 每次执行独立 | 记住所有历史 |
| 无法学习 | 从结果中优化 |
| 重复错误 | 记住成功模式 |
