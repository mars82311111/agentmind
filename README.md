# AgentMind 🧠

> 让AI Agent拥有记忆，持续学习，自动赚钱

**不是**另一个MoneyPrinter复制品。
**是**会学习的自动化赚钱系统——记住历史，持续优化。

## 核心差异

| 传统自动化 | AgentMind |
|-----------|-----------|
| 每次执行独立 | SuperMemory记住所有历史 |
| 无法从错误中学习 | 持续优化策略 |
| 重复同样的错误 | 记住成功模式 |

## 三大模块

### 🐦 Twitter Agent
- 自动发推
- 记忆什么内容表现好
- 分析 engagement 数据
- 智能决定发帖频率

### 📧 Outreach Agent  
- Cold Email 自动发送
- 记忆联系过谁（避免重复）
- 分析什么类别商家最容易成功
- 优化话术

### 🧠 SuperMemory Layer
- 跨模块共享记忆
- 从结果中持续学习
- 记住偏好和模式
- 让AI像真实团队一样协作

## 技术栈

- Python 3.12+
- SuperMemory（持久化记忆系统）
- opencli（Twitter API）
- 24/7 CRON调度

## 快速开始

```bash
git clone https://github.com/mars82311111/agentmind.git
cd agentmind
python src/scheduler.py
```

## 工作原理

```
每次行动 → 记忆存储 → 持续学习 → 优化下次行动
    ↑                                      ↓
    └──────── 记住成功 ──────────→ 重复成功模式
```
