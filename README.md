# AgentMind 🧠

> **让AI Agent拥有记忆，像真实团队一样持续学习**

不是另一个自动化脚本——是一个能**记住一切、持续优化、24/7运行**的智能Agent系统。

[⭐ Star us on GitHub](https://github.com/mars82311111/agentmind) | [🚀 Live Demo](#demo)

---

## ✨ 核心功能

### 🐦 Twitter Agent
- 自动发推，智能频率控制
- **记住**什么内容表现最好
- 从engagement数据中学习
- 持续优化内容策略

### 📧 Outreach Agent  
- 自动联系潜在客户
- **记住**联系过谁，避免重复打扰
- **记住**哪种类型商家最容易回复
- 持续优化话术

### 🧠 SuperMemory Layer
- 所有行动都被记忆
- 跨模块共享上下文
- 从成功和失败中学习
- **永不忘记**你在做什么

## ⚡ 工作原理

```
用户指令 → Agent执行 → 结果记忆 → 持续学习 → 优化策略
                                              ↑
                    ┌─────────────────────────┘
                    ↓
              记住成功模式
              避免重复错误
```

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/mars82311111/agentmind.git
cd agentmind

# 运行（需要配置opencli）
python src/scheduler.py

# 查看记忆数据
ls ~/.money-printer-memory/
```

## 📊 差异化

| 传统方案 | AgentMind |
|---------|-----------|
| 每次执行都是新的 | SuperMemory记住一切 |
| 无法从错误中学习 | 持续优化策略 |
| 重复同样的失败 | 记住成功模式 |
| 孤立的任务 | 跨模块共享记忆 |

## 💡 为什么需要AgentMind？

- AI Agent需要记忆能力才能真正有用
- 单次执行的任务没有积累
- 需要一个系统记住**你**的业务 context

## 📦 技术栈

- Python 3.12+
- SuperMemory 持久化记忆
- opencli（社交媒体API）
- 24/7 调度器
